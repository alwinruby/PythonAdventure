# The three week course at Step Function

A course to bring absolute beginners to writing one hundred line programs
in three weeks.

### Day - 1
### Material Covered

- Variable
- Basic Python built-in function (print,len,type)
- Python Data Type
   - Numeric (float, int)
   - Sequence (string, list)
- Syntax  and Style


### Material Covered

- Logic
  - Boolean Value
  - Boolean Expression
  - Logical Operators
  - Chaining Logical Operators
  - Python Truthiness

- Control Flow
  - Conditional Statement
  - For Loop
  - While Loop

- Exercise:
 - Guessing Game
 - Rolling the dice
